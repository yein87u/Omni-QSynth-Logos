# Omni-QSynth-Logos

---

## 簡介
Logos平台是一個以 React 與 FastAPI 為核心建構的量子電路優化實驗平台，提供從資料輸入、電路生成到結果驗證的端到端流程。使用者可在直覺化介面中建立或匯入功能性真值表（Input ↔ Target Mappings），系統便會自動進行量子布林電路的合成與優化。平台不僅提供演算法結果，更透過即時視覺化技術，讓使用者能觀察電路逐步生成的過程，提升演算法的可理解性與可探索性。

---

## Logos平台的主要特色如下： 
- **直覺化映射輸入**：使用者可直接在平台上建立或匯入真值表，無需撰寫複雜程式碼，即可開始進行電路合成。 
- **SSE 即時電路渲染**：後端透過 Server-Sent Events (SSE) 持續回傳演算法的中間結果，前端即時接收事件流並動態生成量子邏輯閘，以動畫方式呈現電路從無到有的合成過程，突破傳統演算法僅提供最終結果的黑箱限制。 
- **高品質電路產出**：在演算法設計上，平台採用混合策略：先將既有量子布林電路合成方法轉換為可搜尋的表示，以降低搜尋空間，再透過最佳化在可行解集合中尋找高品質解，進而減少冗餘邏輯閘與輔助位元，產生更精簡的量子電路。 
- **無縫對接驗證環境**：合成完成後，平台可將電路匯出為 OpenQASM 格式，能直接在IBM Quantum 平台匯入檔案，亦或是連接其他量子電腦平台，確保電路邏輯的正確性。 

---

## Logos平台的主要使用對象： 
- **量子演算法研究員 / 工程師**：用於快速驗證布林函數的量子電路實作。 
- **量子 EDA (電子設計自動化) 開發者**：作為量子晶片邏輯閘層級合成的參考工具。 
- **教育機構與學生**：透過視覺化過程，直觀了解量子電路的運作與邏輯閘合成原理。
